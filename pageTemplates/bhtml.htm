All About BHTML

BHTML is a special subset of standard HTML (Hyper-text markup language) code which gives our users the power to use HTML formatting commands, while ensuring the security and safety of our members.

Basic Formatting

As in HTML, text is formatted by surrounding text with format codes. Different codes exist for bold, italic, underline, strikethrough, and others. Below, you will find a table as reference that you can use in formatting your text.

Description Code Result
Bold Text <b>text</b> text
Strong Text <strong>text</strong> text
Italic Text <i>text</i> text
Emphasized Text <em>text</em> text
Underlined Text <u>text</u> text
Strikethrough Text <strike>text</strike> text
Blinking Text <blink>text</blink> text

Also note that the above "tags" can be nested to produce a combined effect. For example, <b><i>text</b></i> will result in text.

To embed a < or a > character, use &lt; for the < and &gt; for the >.

Smileys and Arrows

BHTML supports 40 standard smiley codes, and the parser will replace smiley codes with the appropriate smiley image. For brevity, the codes are not listed here, but you will eventually come to terms with them as you see your work's output.

Arrow images can be created by using -->, <--, <downarrow>, and <uparrow>.

Font Formatting

Text can also be formatted using fonts, sizes, and colors. To do this, you will need to create a font tag. Font tags are slightly more complex than basic formatting tags, but they are also slightly more powerful. Here is how you create a font tag:

  • Start with <font
  • Do you want to specify a size?
    • If yes: add size="#" to your tag, replacing the # with a number from 1 to 7 inclusive, 1 being the smallest and 7 being the largest font size
  • Do you want to specify a font face?
    • If yes: add face="#" to your tag, replacing the # with the font name you want to replace
    • Note: if the end user does not have the font you specified, he or she will merely see a system default font such as Times New Roman.
  • Do you want to specify a font color?
    • If yes: add color="#" to your tag, replacing the # with either the name of the color or its hexadecimal equivalent code (more info).
  • End your font tag with a >
  • Close your font tag with a </font> when you want to stop applying your new font settings.

Important: Always make sure you end with a </font> when you are finished. Otherwise, your font settings might escape the affected area. We do have precautions in place to prevent this from happening, but they might not always work perfectly.

Tables

Tables can be easily created by using the following tags. Attributes of the table-related tags are fully supported.

Description Code
Table Root Tag <table>...</table>
Row Element <tr>...</tr>
Cell Element <td>...</td>
Header Cell <th>...</th>

For more information on creating HTML tables, please see W3C: Tables in HTML Documents (external link; opens in new window).

Images

Images can be added to your pages, but must be stored on your own webspace or in your photo album. We do not support the BHTML embedding of photos from your photo album. Syntax to create an image reference is:

<img src="http://address">

Be sure to replace the contents of the SRC attribute with the full URL to the image you wish to include. As a best practice, please avoid including very large images on MeTeen.com pages. Please also avoid linking to slow servers, as they tend to bog down the loading of pages.

Forbidden Tags

To ensure the safety and security of all our users, the following tags are disallowed and will not be rendered as HTML code:

  • IFRAME
  • FRAME
  • BODY
  • HTML
  • HEAD
  • SCRIPT
  • DIV
  • BUTTON
  • STYLE
  • INPUT
  • FORM
  • TEXTAREA
